Projeto elaborado para a disciplina de Banco de Dados 2 da UTFPR-CP

➡️ O projeto consiste em um sistema de banco financeiro, até o momento ele consegue fazer transações e buscar.

    ✅ REACT
    ✅ HTML
    ✅ CSS
    ✅ POSTGREE
